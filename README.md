# SCOPE INDIA Website Clone

Exact replica of SCOPE INDIA training institute website.

## Installation

1. Install dependencies:
   pip install -r requirements.txt

2. Run application:
   python app.py

3. Open browser:
   http://localhost:5000

## Admin Panel
http://localhost:5000/admin

## Features
- Exact color scheme (Navy blue & Golden yellow)
- Registration form with database
- All pages (Home, Courses, Placements, About, Contact)
- Responsive design
- Working backend with SQLite
